## Live WYSIWYG edit acceptance checklist

Use this checklist before considering on-page editing complete:

1. **Inline mode handshake**
   - Open Content Pages in `onpage` workspace mode and toggle inline edit on/off.
   - Confirm `CMS_INLINE_MODE` messages are reflected in preview behavior (editable overlays appear/disappear).
2. **Selection and highlight sync**
   - Select fields/components in admin and verify preview highlight/scroll sync is stable.
   - Trigger on-page selection from preview and verify editor focus/path sync.
3. **Autosave cadence**
   - Modify field values and component props while inline mode is enabled.
   - Confirm autosave schedules and persists draft changes without duplicate/conflicting saves.
4. **Permission and publish constraints**
   - Verify inline edit is disabled for published-only states where mutation is not allowed.
   - Verify forbidden users receive blocked states/messages instead of silent failures.
5. **Regression gates**
   - Run `pnpm --filter @contenthead/admin test`.
   - Run `pnpm test:health`.
